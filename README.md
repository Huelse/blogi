# Blogi

一个高度个性的个人博客，支持主题、插件和AI总结

## 技术栈

Server：JDK 25, Spring Boot 4, PostgreSQL, HikariCP, MyBatis-Plus, Lombok, Redis, Spring Security 7, JWT

Client: Vue 3, Pinia, Nuxt, TypeScript, Milkdown, Tailwind, Element Plus (管理后台)

Docker Compose 启动前后端及其服务
